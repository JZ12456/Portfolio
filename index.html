Starcraft AI

Overview:StarCraft 2 bot built using the BurnySC2 Python API. Alows the simple rule based ai to bbeat all 3 races; Changes build orders when facing different races

Features:

  • Race Strategy Module: Runs pro build orders for the opponents race.
  • MacroManager: Transitions, Macro cycle and Production
  • MicroManager: Target prioritization, retreat logic based on DPS and medivac drop AI.
  • Stable API: Built on BurnySC2’s library, which wraps Blizzard’s official SC2 client API.

How to Run:
Unzip the project, install dependencies, and execute:
python terran_bot_v2.py --race Protoss --difficulty Hard
